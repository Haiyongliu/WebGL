## 2. How to run this demo in ubuntu.
- install npm package
- ```npm install http-server ```,根目录下是否有package.json文件，没有则添加该文件，编辑写入
```    
{
    "scripts": {
        "start": "http-server -a 0.0.0.0 -p 8000"
    }
}
```
- ```npm start```
- 打开浏览器，输入提示网址，例如 http://127.0.0.1:8000/index.html
